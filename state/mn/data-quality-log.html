November 27, 2020

Variables affected:

  • All Metrics

Tags: State Anomalies State Data Note

On November 26, 2020, Minnesota announced on their [COVID-19 page](https://www.health.state.mn.us/diseases/coronavirus/situation.html) that there would be no update to their data on November 26, 2020 due to the Thanksgiving holiday. Because Minnesota’s data is “current as of 4 p.m. the previous day”, this change in their reporting schedule results in the data reported for November 27, 2020 being the data that would normally have been reported on November 26, 2020.

March 6, 2020 to June 13, 2020

Variables affected:

  • positiveCasesViral

In a previous backfill that took place on 6/14, we updated Minnesota's Positives column using state-provided historical data. Minnesota's positives are confirmed cases, so when we did so, we should have also updated the Positive Cases (PCR) column to match what we put in Positives. This has now been done for 6/13 and earlier (the data after that is fine because the Data Entry pub shifts have been setting both correctly since the backfill).
