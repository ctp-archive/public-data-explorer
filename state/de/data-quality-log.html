March 11, 2020 to October 28, 2020

Variables affected:

  • totalTestEncountersViral
  • positiveTestsViral

On October 29, 2020, we moved the data in our **Total Tests (PCR)** field for Delaware to the **Total Test Encounters (PCR)** field after learning that it represents encounters from Delaware's state health department. However, Delaware appears to include antigen tests in its totals, so if an individual is tested using two different test types on the same day, it will get counted twice. Also as of October 29, 2020, Delaware's `totalTestResults` field represents encounters. 200k overall increase in TotalTestResults, 260 day-over-day TotalTestResults increase.

April 29, 2020 to June 10, 2020

Variables affected:

  • positiveCasesViral

Tags: CTP API / Logic Changes

Between April 29, 2020 and June 10, 2020, we recorded Delaware's case metric in our **Confirmed Cases** field as well as the **Cases (confirmed plus probable)** field. When they began reporting confirmed and probable cases separately on June 11, 2020, it became clear that Delaware's case total had included probable cases. We removed the **Confirmed Cases** values from our time series on July 18, 2020.
