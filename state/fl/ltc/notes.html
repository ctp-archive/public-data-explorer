Notes

Florida reports long-term care COVID-19 data for Skilled Nursing, Assisted Living and Intermediate Care Facilities. CTP puts Intermediate Care Facility in the Lumped or Other facilities category in our data set.Florida reports cumulative and outbreak data. Staff and resident cases and deaths are reported as combined or separately, depending on the metric.

Florida combines staff and resident data when reporting cumulative cases and deaths.

Florida provides outbreak data for resident cases and staff cases separately. Florida’s outbreak data for cases reflects information available for current residents and staff with cases as of yesterday’s date.

Florida reports facility-level data in two spreadsheets, one for outbreak cases and one for cumulative deaths. Both spreadsheets report staff and residents separately. The outbreak case data includes facility type but the facility-level death data does not. Florida does not provide facility-level cumulative case data.

  • August 6, 2020: Florida’s reported cumulative resident deaths decreased from the previous data collection on July 30, 2020.
  • October 15, 2020: Florida hasn't updated death data since October 2, 2020. Data is carried over.
  • December 3, 2020: Florida publishes long-term care deaths in a facility-level sheet and also in a daily state report. The CTP long-term care team recently became aware of this cumulative death data on the state report. This number varies from the facility-level data and is released more regularly than the facility level sheet. On this day, CTP backfilled cumulative resident and staff death data going back weekly to May 28, 2020.
  • December 17, 2020: Florida’s death data in the daily state report combines staff and resident deaths. State reported cumulative staff deaths previously gathered were removed historically as they are included in the staff/resident deaths combined.

Source data for cumulative staff/resident cases is in the “County Report” on FL’s COVID-19 page. Source data for cumulative staff/resident deaths is in the “See State Report” link on FL’s COVID-19 page. Source data for facility-level outbreak case data is in FL’s ltcf-case-reports archive. Source data for facility-level cumulative death data is in FL’s  ltcf-death-reports archive.

Facility notes

The state provides facility-level cumulative data. The state provides facility-level outbreak data. Cumulative facility-level data was first reported by the state on 2020-05-08. Outbreak facility-level data was first reported by the state on 2020-05-08.

Because each state reports facility-level data differently, CTP has produced a number of scripts to make the dataset as standardized and comprehensive as possible. This state uses the following scripts: special FL scripts, post_processing, standardize_data, collapse_facility_rows_no_adding. More information on each of these scripts can be found in our Airtable Facility Script Annotations. (https://airtable.com/shr99Ep06cplrcVg6)
