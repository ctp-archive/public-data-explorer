Notes

Wisconsin reports long-term care data for Skilled Nursing Facilities and Long-Term-Care Facilities. Long-term-care facilities include Skilled Nursing Facilities, Assisted Living Facilities, Community-based Residential Facilities, and Residential Care Apartment Complexes. COVID Tracking Project puts Skilled Nursing Facilities data in the Nursing Home category, and Long-Term-Care data in the Lumped or Other category in our dataset. Wisconsin reports cumulative and outbreak data. Wisconsin reports facility-level data. Wisconsin reports staff and resident cases and staff and resident deaths as combined.

Wisconsin provides cumulative totals for resident cases and resident deaths and number of facilities with a case for all long-term-care facilities. Because Long-Term-Care Facilities includes Nursing Homes, COVID Tracking Project subtracted the total of Nursing Home Facilities from the total number of Long-Term-Care Facilities to prevent double counting.

Wisconsin provides Nursing Home outbreak data of the number of facilities with a case. Only the names of Nursing Homes with an active outbreak are provided.

Wisconsin provides facility-level data. The state reports the names of outbreak nursing homes. Wisconsin defines an outbreak (called an “investigation”) in these facilities as a single confirmed case occurring in a facility in a 28-day period. An outbreak ends when 28 days have passed with no new confirmed case.    Wisconsin releases facility names of outbreak Nursing Homes.

  • June 25, 2020: CTP started gathering data for number of facilities with active outbreaks in long-term care

Facility notes

The state does not provide facility-level data.

Because each state reports facility-level data differently, CTP has produced a number of scripts to make the dataset as standardized and comprehensive as possible. This state uses the following scripts: post_processing, standardize_data, close_outbreaks. More information on each of these scripts can be found in our Airtable Facility Script Annotations.
